# 🎮 Tic Tac Toe

A modern and interactive Tic-Tac-Toe game built with Python and Tkinter. Featuring a clean user interface, live score tracking, turn indicators, and smooth gameplay for an enjoyable desktop gaming experience.

## ✨ Features

- 🎮 Interactive Tic-Tac-Toe Gameplay
- 🏆 Live Scoreboard
- 🎯 Turn Indicator
- 🔄 New Match Button
- 📊 Reset Score Option
- 🖥️ User-Friendly GUI
- ⚡ Lightweight & Fast
- 🐍 Built with Python and Tkinter

## 🎯 How to Play

1. Player **X** starts the game.
2. Players take turns placing their marks.
3. Get three marks in a row, column, or diagonal to win.
4. If all cells are filled without a winner, the game ends in a draw.
